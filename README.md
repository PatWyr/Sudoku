# Sudoku
Sudoku game 
Sudoku game with access to database
Used technologies Gluon,SQLite,Maven,JavaFX
Gra utworzona na zliczenie przedmiotu Programowanie Komponentowe
Aplikacja pozwala na generowanie plansz o różnym stopniu trudności oraz zapisywanie postepów
gry na utworzoną baze danych SQLite. Gra zawiera własną kontrole błędów oraz pełne graficzne menu.
